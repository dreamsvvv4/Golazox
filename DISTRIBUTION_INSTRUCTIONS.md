## ⚡ Installation Instructions for Recipients

### Windows (Recommended)
1. Extract `Ultra-KibanaDownloader-v1.0-2025-07-10.zip`
2. Open the extracted folder
3. Navigate to `Requirements` folder
4. Double-click `install_and_start_elasticlogana_WINDOWS.bat`
5. Application will launch automatically

### Linux/Mac
1. Extract `Ultra-KibanaDownloader-v1.0-2025-07-10.zip`
2. Open terminal in extracted folder
3. Run: `chmod +x Requirements/install_and_start_elasticlogana_LINUX.sh`
4. Run: `./Requirements/install_and_start_elasticlogana_LINUX.sh`
